ПРОСТАЯ СХЕМА ГЛАВНОГО ЭКРАНА

1. Переход на главный экран
   - Пользователь успешно вошел в систему
   - Скрывается экран авторизации
   - Показывается главное меню

2. Главный экран
   - Отображаются:
     • баланс
     • трофеи
     • кнопка статистики
     • кнопка арены
     • кнопка спинов
     • кнопка турнира
   - Пользователь видит профиль и доступные режимы

3. Действие: Арена
   - Пользователь нажимает [В бой]
   - Отправка: начать матч или подключиться к игре
   - Переход: экран игры

4. Действие: Спины
   - Пользователь нажимает [Испытать удачу]
   - Отправка: выбрать режим спина
   - Переход: экран спинов / рулетки

5. Действие: Турнир
   - Пользователь нажимает [Запустить Турнир]
   - Отправка: запустить турнирный режим
   - Переход: экран турнира

6. Действие: Статистика
   - Пользователь нажимает [Статистика]
   - Отправка: запрос /api/user/stats?login=...
   - Отображение: история игр, % побед, макс. выигрыш

7. Выход назад
   - При любом неудачном действии остается на главном экране
   - Пользователь может выбрать другую кнопку

КРАТКИЙ ПОТОК

Success
  ↓
Main screen
  ↓
[Арена] → Start arena → Game screen
  ↘
   or
[Спины] → Start spin → Spin screen
  ↘
   or
[Турнир] → Start tournament → Tournament screen
  ↘
   or
[Статистика] → Load stats → Stats screen

Ключевые блоки
- Success
- Main screen
- Balance / Trophies
- Arena
- Spins
- Tournament
- Statistics
- Error / Stay on main

Эта схема описывает только блок главного экрана и основные переходы. Если нужно, могу сделать её ещё короче и в виде таблицы.
